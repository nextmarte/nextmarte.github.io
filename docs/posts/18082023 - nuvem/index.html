Você está com problemas para instalar as bibliotecas e IDE’s do seu bootcamp porque seu PC é fraco? Eu trago a solução pra maior parte dos seu problemas, o GitHub codespaces, com ele você pode criar um ambiente de desenvolvimento para qualquer stack de graça. Eu escrevi um artigo para o meu blog com foco no R com QUARTO, mas você pode fazer o mesmo para qualquer linguagem. O codespaces é um container que roda uma distro linux (UBUNTU) e já vem com o python e o jupyter instalado. Dá uma olhada no meu blog post para saber como acessar: CLIQUE AQUI PARA ACESSAR!.

Além do codespaces existem outras opções que vou listar aqui, todas baseadas no navegador.

Jupyter - Possui o Jupyter notebook e o ambiente labs, suporta mais de 40 linguagens além de R, Python, C, JavaScript.

Posit Cloud- Ambiente de desenvolvimento na núvem da POSIT usando a IDE RStudio, suporte: R, Quarto, C, C++, Markdown, HTML, CSS, JavaScript, D3, Python, Shell, SQL, Stan, Text, R Sweave, R HTML e outras…

Colab - Solução do Google para desenvolvimento colaborativo na núvem, suporta: Python, R, Julia, Swift, JavaScript, TypeScript, Shell, Markdown

Uma diferença dessas opções para o codespaces é que você pode acessar o espaço criado no GitHub a partir do VScode instalado na sua máquina, além disso como é uma distro Linux, você pode instalar qualquer linguagem pela linha de comando. Se você conhece outras soluções baseadas na nuvem que oferecem serviço gratuito, coloca nos comentários e vamos aumentar a lista.
